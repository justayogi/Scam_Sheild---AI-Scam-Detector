# Scam_Sheild---AI-Scam-Detector
Paste a message/email → AI tells you if it’s a scam.
🛡️ ScamShield – AI-Powered Scam Detector
🚀 Overview

ScamShield is an AI-powered web application that helps users identify potential scams in messages, emails, or text.

Users can paste suspicious content and instantly receive:

A risk score (Low / Medium / High)
A clear explanation of why it may be a scam
Actionable advice on what to do next

With phishing and online scams on the rise, ScamShield aims to make digital spaces safer and more transparent.

🎯 Problem

Millions of people fall victim to scams every year due to:

Lack of awareness
Sophisticated phishing techniques
Messages that appear legitimate

ScamShield provides a simple, accessible tool to help users make safer decisions online.

💡 Solution

ScamShield uses AI to:

Analyse text patterns
Detect suspicious language
Identify common scam tactics (urgency, threats, impersonation)
🧠 Features
🔍 Paste and analyse suspicious messages
⚠️ Risk classification (Low / Medium / High)
📖 Explanation of detected scam indicators
💬 User-friendly interface
